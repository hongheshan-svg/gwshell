Key differences in the CC Switch style

  • Navigation: Text tabs replaced with icon-only toolbar — app pills on left, action icons on right
  • Provider avatars: Rounded circle (not square) with colored background + initial letter
  • Provider cards: No gradient overlay, lighter highlight for active item
  • Add button: Orange circle + button in top toolbar (not bottom bar)
  • Overall: Cleaner, more compact, less chrome
