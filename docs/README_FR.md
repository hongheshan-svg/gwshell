## ✨ Fonctionnalités

### Implémentées ✅

| Catégorie | Fonctionnalité | Description |
|-----------|---------------|-------------|
| **SSH** | Connexions SSH multi-onglets | Se connecter à plusieurs serveurs simultanément |
| **SSH** | Authentification mot de passe / clé / MFA | Mot de passe, clé privée, TOTP/2FA |
| **SSH** | Hôte de saut (-J) | Connexions proxy via bastion |
| **SSH** | Proxies SOCKS4/5, HTTP | Router les connexions SSH via des proxies |
| **SSH** | Redirection de ports | Redirection de port local (tunnel SSH) |
| **SSH** | Latence de connexion (ping) | Mesure de latence TCP en temps réel |
| **SFTP** | Navigateur de fichiers | Parcourir les systèmes de fichiers distants |
| **SFTP** | Téléverser / Télécharger | Transférer des fichiers entre local et distant |
| **SFTP** | Éditeur de texte intégré | Modifier des fichiers distants directement |
| **Terminal** | Terminal local | PowerShell, CMD, Bash, Zsh, etc. |
| **Terminal** | Écran partagé | Dispositions 1×1, 1×2, 2×2, 2×3, 2×4 |
| **Série** | Console port série | Connexion aux appareils COM/ttyUSB |
| **AI** | Gestion des fournisseurs | Configurer et changer les fournisseurs AI CLI |
| **UI** | Thème sombre / clair | Détection automatique du système |
| **UI** | i18n (English, 中文) | Interface bilingue complète |
| **Données** | Persistance SQLite | Stockage local des sessions et paramètres |
| **Mise à jour** | Mise à jour automatique | Vérification et installation des mises à jour dans l'app |
